### 1. 小数精度的价值
```css
/* 错误：使用四舍五入 */

gap: 172px;    /* 原始171.47px */

/* 正确：保留小数精度 */
height: 36.55px;  /* 精确 */
gap: 171.47px;    /* 精确 */
```

### 2. 透明度层次
```css
/* 6级透明度创造丰富的视觉层次 */
```
